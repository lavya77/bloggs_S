eyezs - A Modern Django Blog Platform

eyezs is a full-featured, modern blog platform built with Django and styled with Tailwind CSS. It provides a clean, professional, and intuitive user interface for writers and readers, inspired by modern social media platforms.

## Features

    User Authentication: Secure user registration, login, and logout functionality.

    Profile Management: Users can create and update their profiles with a bio and a profile picture.

    CRUD for Posts: Authenticated users can create, read, update, and delete their own blog posts.

    Interactive Content: Users can "like" posts and leave comments.

    Instagram-Style UI: A polished user interface with features like collapsible comment sections and interactive buttons.

    Notifications: Users are notified when someone likes or shares their post.

    Search Functionality: A search bar to easily find posts by title.

    Responsive Design: A mobile-friendly layout that looks great on all devices.

## Technology Stack

    Backend: Django, Python

    Frontend: HTML, Tailwind CSS, Alpine.js (for dropdowns and interactivity)

    Database: SQLite3 (default, can be swapped for PostgreSQL, etc.)
